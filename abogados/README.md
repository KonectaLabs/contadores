## Tesis del funnel

No vender "IA para abogados", "marketing legal" ni una herramienta.

Vender consultas de casos que dejan honorarios, entrando al WhatsApp del estudio:

- sucesiones con inmueble;
- despidos con telegrama;
- amparos urgentes;
- accidentes con documentacion.
